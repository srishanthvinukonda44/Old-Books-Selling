# 📚 Old Books Selling Platform

A responsive web application that allows users to buy and sell old books online.  
The platform focuses on simple navigation, clean UI/UX and a smooth experience for book lovers and sellers.

---

## 🚀 Features
- User-friendly interface for browsing and listing books
- Categories and search functionality for quick access
- Responsive design (works on desktop and mobile)
- Secure form handling for sellers to add book details
- Modern UI with HTML, CSS and JavaScript

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Design:** Responsive layout with clean UI/UX principles

---

## 📂 Project Structure
